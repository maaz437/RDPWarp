# RDPWarp
To allow multiple concurrent session on windows OS (Enable Multiple RDP Sessions)

##################################################################################
Step 1:
  extract rdpwup_8

Step 2:
  Run rdpwup

Step 3:
  Click on update

Step 4:
  Run install.bat

Step 5:
  Run update.bat

Step 6:
  Run RDPConf.exe

Step 7:
  Click on OK

Step 7:
  Run RDPWInst.exe
